# Página Pessoal – Desenvolvedor em Formação

Projeto simples desenvolvido com HTML e CSS com o objetivo de praticar
estruturação de páginas web e estilização básica.

## 🛠 Tecnologias utilizadas
- HTML5
- CSS3
- VS Code
- Codex

## 📄 Descrição
Página pessoal contendo informações básicas como nome, formação,
habilidades em desenvolvimento e contato.

## 🚀 Ferramentas e Aprendizado Contínuo
Utilizei ferramentas como VS Code e Codex para otimizar meu fluxo de trabalho
no desenvolvimento, focando no aprendizado contínuo e na melhoria constante
das minhas habilidades de programação.

## 🎯 Objetivo do projeto
Projeto criado para fins de aprendizado e prática inicial em desenvolvimento web.
